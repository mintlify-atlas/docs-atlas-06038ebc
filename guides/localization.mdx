## Using a Built-in Language

Simply specify the language:

```yaml
locale:
  language: german
```

That's it! All date formatting, month names, and other text will automatically use German.

## Locale Fields

The locale configuration includes:

```yaml
locale:
  language: english
  last_updated: Last updated in      # For top note
  month: month                        # Singular
  months: months                      # Plural
  year: year                          # Singular
  years: years                        # Plural
  present: present                    # For ongoing positions
  month_abbreviations:                # Short month names
    - Jan
    - Feb
    - Mar
    - Apr
    - May
    - June
    - July
    - Aug
    - Sept

    - Dec

```

## Where Locale Affects Your CV

### Date Formatting

Locale controls how dates appear:

```yaml
cv:
  sections:
    experience:
      - company: Tech Corp
        start_date: 2020-05
        end_date: 2023-08
```

**English:** May 2020 – Aug 2023  
**German:** Mai 2020 – Aug 2023  
**Spanish:** Mayo 2020 – Ago 2023

### Time Spans

For ongoing positions or when `show_time_spans_in` is enabled:

```yaml
end_date: present
```

**English:** 3 years 6 months  
**German:** 3 Jahre 6 Monate  
**Spanish:** 3 años 6 meses

### Top Note

The "last updated" message at the top of your CV:

**English:** Last updated in March 2026  
**German:** Zuletzt aktualisiert im März 2026  
**Spanish:** Última actualización en marzo de 2026

## Customizing Locale

### Partial Override

Start with a language and override specific fields:

```yaml
locale:
  language: english
  present: now            # Override just this field
  month: mo
  months: mos
```

Now ongoing positions show "now" instead of "present", and time spans show "3 yrs 6 mos".

## Phrases

Some locales support custom phrases for combining text:

```yaml
locale:
  language: english
  phrases:
    degree_with_area: DEGREE in AREA
```

This controls how education entries display degree and area:

**Default:** PhD in Computer Science  
**Custom:** PhD - Computer Science (if you set `degree_with_area: DEGREE - AREA`)

## Complete Examples

```yaml John_Doe_EN.yaml
  location: San Francisco, CA

        end_date: present
```

## Date Format Customization

Control how dates appear using templates in the `design` section:

```yaml
design:
  templates:
    single_date: MONTH_ABBREVIATION YEAR          # "Jan 2020"
    date_range: START_DATE – END_DATE             # "Jan 2020 – Dec 2023"
    time_span: HOW_MANY_YEARS YEARS HOW_MANY_MONTHS MONTHS  # "3 years 6 months"
```

<Tabs>
  <Tab title="Standard">
    ```yaml
    templates:
      single_date: MONTH_ABBREVIATION YEAR
      date_range: START_DATE – END_DATE
    ```
    Output: May 2020 – Aug 2023
  </Tab>

  <Tab title="Full Month Names">
    ```yaml
    templates:
      single_date: MONTH_NAME YEAR
      date_range: START_DATE – END_DATE
    ```
    Output: May 2020 – August 2023
  </Tab>

  <Tab title="Year Only">
    ```yaml
    templates:
      single_date: YEAR
      date_range: START_DATE – END_DATE
    ```

  </Tab>

  <Tab title="Numeric">
    ```yaml
    templates:
      single_date: MONTH_IN_TWO_DIGITS/YEAR

    ```
    Output: 05/2020 – 08/2023

## Tips for Localization

<AccordionGroup>
  <Accordion title="Consistency">
    - Use the same language throughout (content and locale)
    - Ensure section titles match the locale language
    - Translate all text, not just dates
  </Accordion>

  <Accordion title="Cultural Considerations">
    - Different countries have different CV conventions
    - Some regions prefer photos, others don't
    - Date formats vary (MM/YYYY vs YYYY-MM)
  </Accordion>

  <Accordion title="Testing">
    - Generate the CV and carefully review all dates
    - Check that time spans display correctly
    - Verify "present" appears as expected
  </Accordion>

  <Accordion title="Font Support">
    For non-Latin scripts (Chinese, Japanese, Korean, Russian):
    - Use fonts that support the character set
    - Test that all characters render correctly
    - Consider using Noto Sans or similar universal fonts
  </Accordion>
</AccordionGroup>

## Locale Reference

All available locale fields:

| Field | Type | Description | Example |
|-------|------|-------------|----------|
| `language` | string | Built-in language to use | `english` |
| `last_updated` | string | "Last updated" prefix | `Last updated in` |
| `month` | string | Singular month | `month` |
| `months` | string | Plural months | `months` |
| `year` | string | Singular year | `year` |
| `years` | string | Plural years | `years` |
| `present` | string | For ongoing positions | `present` |
| `month_abbreviations` | list | 12 short month names | `[Jan, Feb, ...]` |
| `month_names` | list | 12 full month names | `[January, ...]` |
| `phrases` | dict | Custom text combinations | See above |
