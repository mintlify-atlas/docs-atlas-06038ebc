RenderCV gives you complete control over your CV's visual appearance. You can customize colors, fonts, spacing, page layout, and more.

## Customization Approach

Start with a theme and override only what you want to change:

```yaml
design:
  theme: classic  # Start with a theme
  colors:
    name: rgb(255, 0, 0)  # Override specific values
```

Or specify everything for complete control:

```yaml
design:
  theme: classic
  page:
    size: a4
    top_margin: 1in
  colors:
    # ... all color options
  typography:
    # ... all typography options
  # ... etc.
```

## Colors

Customize colors for all CV elements:

```yaml
design:
  theme: classic
  colors:
    body: rgb(0, 0, 0)              # Main body text
    name: rgb(0, 79, 144)            # Your name in header
    headline: rgb(0, 79, 144)        # Headline text
    connections: rgb(0, 79, 144)     # Email, phone, links
    section_titles: rgb(0, 79, 144)  # Section headings
    links: rgb(0, 79, 144)           # Hyperlinks
    footer: rgb(128, 128, 128)       # Footer text
    top_note: rgb(128, 128, 128)     # Top note text
```

## Typography

### Font Family

Set fonts for different elements:

```yaml
design:
  typography:
    font_family:
      body: Source Sans 3
      name: Source Sans 3
      headline: Source Sans 3
      connections: Source Sans 3
      section_titles: Source Sans 3
```

**Shorthand:** Apply one font everywhere:

```yaml
design:
  typography:
    font_family: Source Sans 3
```

### Font Size

```yaml
design:
  typography:
    font_size:
      body: 10pt
      name: 30pt
      headline: 10pt
      connections: 10pt
      section_titles: 1.4em
```

Units supported: `pt`, `em`, `cm`, `in`

### Font Styling

Control bold and small caps:

```yaml
design:
  typography:
    bold:
      name: true              # Bold name
      headline: false
      connections: false
      section_titles: true    # Bold section titles
    
    small_caps:
      name: false
      headline: false
      connections: false
      section_titles: false
```

### Text Alignment

```yaml
design:
  typography:
    alignment: justified                          # Body text alignment
    date_and_location_column_alignment: right     # Right column alignment
    line_spacing: 0.6em                          # Space between lines
```

**Alignment options:**
- `justified`: Spreads text across full width
- `justified-with-no-hyphenation`: Full width, no word breaks
- `left`: Left-aligned
- `center`: Centered
- `right`: Right-aligned

## Page Settings

Configure page size and margins:

```yaml
design:
  page:
    size: us-letter
    top_margin: 0.7in
    bottom_margin: 0.7in
    left_margin: 0.7in
    right_margin: 0.7in
    show_footer: true
    show_top_note: true
```

## Header

Customize the header layout:

```yaml
design:
  header:
    alignment: center                  # left, center, or right
    photo_width: 3.5cm
    photo_position: left               # left or right
    photo_space_left: 0.4cm
    photo_space_right: 0.4cm
    space_below_name: 0.7cm
    space_below_headline: 0.7cm
    space_below_connections: 0.7cm
```

### Connection Display

```yaml
design:
  header:
    connections:
      phone_number_format: national    # national, international, or E164
      hyperlink: true                  # Make connections clickable
      show_icons: true                 # Show Font Awesome icons
      display_urls_instead_of_usernames: false
      separator: ''                    # Or: '|', 'â€˘', etc.
      space_between_connections: 0.5cm
```

## Section Titles

Customize section heading styles:

```yaml
design:
  section_titles:
    type: with_partial_line   # Style type
    line_thickness: 0.5pt
    space_above: 0.5cm
    space_below: 0.3cm
```

### Section Title Types

<Tabs>

```yaml
    short_second_row: true
```

## Sections

Control section-level spacing:

```yaml
design:
  sections:
    allow_page_break: true
    space_between_regular_entries: 1.2em
    space_between_text_based_entries: 0.3em
    show_time_spans_in:
      - experience
```

<Note>
  `show_time_spans_in` displays duration calculations like "2 years 3 months" for entries in the specified sections.
</Note>

## Links

Customize hyperlink appearance:

```yaml
design:
  links:
    underline: false
    show_external_link_icon: false
```

## Tips for Great Design

<AccordionGroup>
  <Accordion title="Consistency is Key">
    - Use the same color for all accent elements (name, section titles, links)
    - Stick to 1-2 font families maximum
    - Keep spacing values consistent
  </Accordion>

  <Accordion title="Readability First">
    - Body text should be 10-11pt
    - Line spacing of 0.6-0.7em works well
    - Ensure sufficient contrast between text and background
  </Accordion>

  <Accordion title="Professional Appearance">
    - Avoid bright, saturated colors
    - Use standard margins (0.7-1in)
    - Don't over-customizeâ€”sometimes less is more
  </Accordion>

  <Accordion title="Test Your Design">
    - Print or export to PDF and review carefully
    - Check that all text is readable
    - Verify that important content fits on one page
  </Accordion>
</AccordionGroup>
